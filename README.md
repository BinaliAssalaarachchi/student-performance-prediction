# Student Performance Prediction & Study Schedule Generator

## 📌 Overview
This project predicts students' academic performance using a machine learning model and generates a personalized study schedule.

Developed as part of the BCS Higher Education Qualification (HEQ).

---

## 🏗️ System Architecture
- Frontend: Flutter mobile application
- Backend: Python (Flask/Django)
- Machine Learning Model: Scikit-learn
- Database: MySQL

---

## 🛠 Tools & Technologies
- Python
- Flask / Django
- Scikit-learn
- Pandas & NumPy
- Flutter
- MySQL
- REST APIs

---

## ⚙️ Key Features
- Predict student performance using ML model
- Generate personalized study schedules
- User authentication system
- API-based communication between frontend & backend
- Database storage and retrieval

---

## 🤖 Machine Learning Model
- Algorithm: Decision Tree
- Dataset: Student Exam Performance dataset
- Evaluation: Accuracy-based validation

---

## 📂 Project Structure
- backend → API & ML model
- frontend → Flutter mobile app
- report → Final documentation
